# Agri Smart Hub

## Project Overview
Agri Smart Hub is a frontend web application designed to connect farmers, buyers, and agricultural experts. The platform provides access to smart farming technologies, market opportunities, and climate-related advisory information.

The goal of this project is to demonstrate how web interfaces can support modern agricultural practices and improve accessibility to resources.

---

## Features
- Smart farming technology showcase
- Market access information
- Climate advisory section
- Multi-page navigation (Home, About, Services, Innovations, Contact)
- Clean and structured user interface

---

## Technologies Used
- HTML
- CSS
